安装与校验指南

从验证镜像到制作启动盘，再到全新安装系统的完整流程

01

下载镜像文件

从本站分类页面找到需要的系统版本，点击下载链接获取 ISO 镜像文件。建议使用 IDM 或 FDM 等多线程下载工具以加快下载速度。

本站提供的下载链接均指向第三方镜像站，文件为微软官方原版，未经任何修改。
02

验证哈希校验码

下载完成后，必须验证文件的哈希值是否与本站公布的校验码一致，以确保文件完整且未被篡改。

Windows 命令提示符 (SHA-1)
certutil -hashfile 路径\文件名.iso SHA1
PowerShell (SHA-256)
Get-FileHash 路径\文件名.iso -Algorithm SHA256
比对方法：将命令输出的哈希值与详情页显示的 SHA-1 或 SHA-256 逐位比对。如有任何字符不一致，说明文件已被损坏或篡改，请勿使用。
03

制作安装启动盘

推荐使用 Rufus 或 Ventoy 制作 U 盘启动盘。以下以 Rufus 为例：

  1. 下载并运行 Rufus
  2. 插入 U 盘（至少 8GB），在 Rufus 中选择该 U 盘
  3. 点击「选择」按钮，找到并选择下载的 ISO 镜像文件
  4. 分区类型选择「GPT」，目标系统类型选择「UEFI (非 CSM)」
  5. 点击「开始」，等待写入完成
提示：如使用 Ventoy，只需将 ISO 文件复制到 U 盘即可，无需反复格式化，支持多系统镜像共存。
04

设置 BIOS/UEFI 启动

将制作好的启动 U 盘插入电脑，重启并进入 BIOS/UEFI 设置界面，调整启动顺序。

  1. 重启电脑，连续按 F2 / Del / F12 进入启动菜单
  2. 在启动选项中，将 U 盘设置为第一启动项
  3. 保存设置并退出（通常按 F10）
不同品牌主板进入 BIOS 的按键不同，常见的有 F2、Del、F12、Esc，请根据主板型号确认。
05

安装操作系统

从 U 盘启动后将进入 Windows 安装程序，按以下步骤操作：

  1. 选择语言、时间格式和输入法，点击「下一步」
  2. 点击「现在安装」
  3. 输入产品密钥（如有），或点击「我没有产品密钥」稍后激活
  4. 选择要安装的版本（家庭版/专业版等），注意与密钥匹配
  5. 选择「自定义：仅安装 Windows」进行全新安装
  6. 选择安装分区，可新建/格式化分区
  7. 等待安装完成，系统将自动重启
建议：全新安装前请备份好重要数据。安装完成后，建议先安装所有系统更新，再安装驱动程序。
06

安全校验建议

为确保系统安全，安装完成后请执行以下操作：

  • 立即运行 Windows Update 安装所有安全更新
  • 从硬件厂商官网下载驱动程序，避免使用第三方驱动工具
  • 安装可靠的安全软件（Windows Defender 已足够）
  • 不要运行来源不明的激活工具或破解程序
  • 如系统提示激活，请使用正版产品密钥
重要：本站仅提供原版镜像索引，不提供任何激活工具或破解补丁。请支持正版软件。
