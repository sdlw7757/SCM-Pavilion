安全校验 三步走

为确保你下载的是官方原版镜像，请务必执行以下校验步骤

1. 下载镜像
从本站提供的来源链接下载镜像文件，建议优先选择官方 MSDN 源或已验证的第三方源
2. 验证哈希值
使用 certutil 或 PowerShell 计算下载文件的 SHA-1/SHA-256，与本站公布的校验码逐位比对
3. 制作启动盘
使用 Rufus 或 Ventoy 等工具制作安装启动盘，参考本站安装教程完成系统安装
